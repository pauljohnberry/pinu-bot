API Guide

              

                

                  createRobotFace(canvas, options?)
                  Create a face runtime bound to a canvas.
                

                

                  face.emote(name, options?)
                  Animate to a named emotion.
                

                

                  face.perform(name) / face.transitionTo(state)
                  Run a cinematic sequence or push direct pose overrides.
                

                

                  face.lookAt(x, y)
                  Move gaze using normalized coordinates.
                

                

                  face.lookLeft(amount) / face.lookUp(amount)
                  Directional gaze helpers for quick expressions.
                

                

                  face.smile(amount, { open }) / face.frown(amount, { open })
                  Convenience mouth actions with open or closed variants.
                

                

                  face.pout(amount, { open })
                  Pursed-lip mouth action with optional open depth.
                

                

                  face.blink() / face.wink(side)
                  Trigger quick eye overlays.
                

                

                  face.speak({ intensity, cadence, durationMs })
                  Layer speaking animation on top of the current emotion.
                

                

                  face.perform("bootUp" | "glitch")
                  Play cinematic display transitions.
                

                

                  face.setTheme(...) / face.setStyle(...) / face.setBackgroundFx(...)
                  Switch visual themes, style presets, and mood lighting.
                

                

                  face.setParts(...) / face.setFaceTheme(...)
                  Configure shape language and presets.
                

                

                  face.setMode(...) / face.showSymbol(...) / face.showFace()
                  Swap between face and symbol display modes.
                

                

                  face.eyes() / face.leftEye() / face.rightEye() / face.mouth() / face.nose()
                  Use direct part-control chains for precise adjustments.
                

                

                  face.configure(...) / face.start() / face.stop() / face.render() / face.destroy()
                  Configure and manage the runtime lifecycle.
